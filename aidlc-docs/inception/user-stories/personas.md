# ペルソナ定義

---

## Persona 1: 田中ユウキ（26歳・IT企業営業）

| 項目 | 内容 |
|---|---|
| **年齢** | 26歳 |
| **職業** | IT企業の営業職（3年目） |
| **ライフステージ** | 社会人・一人暮らし |
| **サービスとの関わり方** | ヘビーユーザー（毎日ログイン、コアループを回す） |

### 背景
新卒で入社して3年目。仕事は慣れてきたが、同期が昇進したり資格を取ったりする中で「自分は何もしていない」と感じることが増えた。休日はNetflixを見て終わることが多く、月曜の朝に罪悪感を覚える。SNSで同世代のキラキラした投稿を見るたびに焦るが、結局何も始められない。

### 動機
- 「ダメな自分」を責めるのをやめたい
- 気軽に毎日触れるゲームが好き（ソシャゲ経験あり）
- コレクション要素に弱い（図鑑を埋めたくなるタイプ）

### 行動パターン
- 毎晩の通知でスワイプ報告を欠かさない
- ショップでアイテムを計画的に購入し、親密度を効率よく上げる
- 図鑑コンプリートを目指して様々なダメを試す
- おつかい解禁後は積極的に応える

### このペルソナが検証するストーリー
コアループの全体（スワイプ→コイン→ショップ→贈り物→親密度→手紙→おつかい）

## Persona 2: 佐藤ミホ（34歳・時短勤務ワーキングマザー）

| 項目 | 内容 |
|---|---|
| **年齢** | 34歳 |
| **職業** | メーカー事務職（時短勤務） |
| **ライフステージ** | 既婚・子育て中（3歳の子ども） |
| **サービスとの関わり方** | ライトユーザー（2〜3日に1回、放置も多い） |

### 背景
育児と仕事の両立で毎日がいっぱいいっぱい。「もっとちゃんとした母親にならなきゃ」「仕事も手を抜いてる」と自分を責めがち。子どもが寝た後の30分だけが自分の時間だが、疲れてスマホをぼーっと見て終わることが多い。それすら「時間を無駄にした」と感じてしまう。

### 動機
- 「何もしなかった日」を肯定してほしい
- 長時間の操作は無理。短時間で完結するものがいい
- 癒し系のゆるいゲームが好き

### 行動パターン
- 通知が来ても開かない日が多い（放置ボーナスが蓄積）
- 開いたときにまとめて収穫し、気が向いたらスワイプ
- 途中離脱が多い（疲れて最後までスワイプしない）
- ナマケモノのつぶやきや手紙に癒される

### このペルソナが検証するストーリー
放置ボーナス、途中離脱ボーナス、通知体験、手紙の感情的価値

## Persona 3: 木村ソウタ（21歳・大学生）

| 項目 | 内容 |
|---|---|
| **年齢** | 21歳 |
| **職業** | 大学3年生（文系） |
| **ライフステージ** | 学生・一人暮らし |
| **サービスとの関わり方** | コアユーザー（親密度MAXまで到達し、おつかいを体験する） |

### 背景
授業が少ない曜日は家でゴロゴロしがち。ゲームしたり動画見たりで1日が終わることも多い。別に深刻に悩んでいるわけではないが、「今日も何もしなかったな」とは思う。外に出るのが面倒で、コンビニすら億劫な日がある。友達に誘われれば出かけるが、自分からは動かない。

### 動機
- 暇つぶしとして面白そう
- 「ダメでいい」というコンセプトに共感
- ナマケモノのキャラクターが好き
- ナマケモノに頼まれたら「仕方ないな」と動ける（自分のためには動けないが、相棒のためなら）

### 行動パターン
- 毎日複数回アプリを開く（暇だから）
- スワイプ報告を毎日やり、出現条件を積極的に進める
- 親密度MAXを最速で目指す
- おつかいが届くと「仕方ないな」と言いつつコンビニに行く
- おつかいの会話を楽しむ（ナマケモノとの掛け合い）

### このペルソナが検証するストーリー
おつかい機能の全体（解禁→受信→実行→報告→AI応答→森の日記）
