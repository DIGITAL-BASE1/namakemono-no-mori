# User Stories 生成プラン

## 概要
ナマケモノの森のMVP v1（1.5日体験）をユーザーストーリーとペルソナに変換する。

**ユーザーストーリーとは**: ソフトウェア開発の手法。「誰が」「何をしたくて」「なぜ」を1文で書き、それぞれに「完成の定義（受け入れ基準）」を付ける。requirements.mdの14機能を、開発・動画撮影で使える具体的な単位に分解するのが目的。

---

## Part 1: Planning — 質問と方針決定

### Q1. ペルソナの粒度

MVP v1のデモシナリオは「1人のユーザーの1.5日体験」です。ペルソナ（＝デモの主人公像）はどの程度定義しますか？

- A) 1人だけ（デモの主人公）。シンプルに1人の体験を追う
- B) 2〜3人（メインペルソナ + サブペルソナ）。異なるユーザー像で多角的に
- C) ペルソナは不要。ストーリーだけあればいい
- X) Other (please describe after [Answer]: tag below)

[Answer]: A

### Q2. ストーリーの分解アプローチ

ストーリーをどの軸で分解しますか？

**A) 時系列型** — デモシナリオの時間軸で並べる：
```
Epic: Day1初回起動
  - アプリを開いてウェルカム画面を見る
  - 空の森を見る
  - シルエットだらけの図鑑を見る
Epic: Day1夜
  - 通知を受け取る
  - スワイプで報告する
  ...
```

**B) 機能ベース** — システムの機能単位で並べる：
```
Epic: スワイプ報告
  - カードをスワイプする
  - 途中離脱する
Epic: 図鑑
  - シルエットを見る
  - ナマケモノを登録する
  ...
```

**C) ハイブリッド** — 時系列をEpicにして、中身を機能単位で分解：
```
Epic: Day1初回起動（時系列）
  - オンボーディングを完了する（機能: オンボーディング）
  - 空の森を見る（機能: 森画面）
  - 図鑑のシルエットを見る（機能: 図鑑）
Epic: Day1夜（時系列）
  - 通知を受け取る（機能: 通知）
  - スワイプで報告する（機能: スワイプ報告）
  ...
```
ハイブリッドは「デモの流れ」と「実装の単位」の両方が見える。

- A) 時系列型
- B) 機能ベース
- C) ハイブリッド
- X) Other (please describe after [Answer]: tag below)

[Answer]:B

### Q3. ストーリーの粒度

1つのユーザーストーリーの大きさはどの程度を想定しますか？

**A) 大きめ（シーン単位）** — 5〜8ストーリー：
```
「ユーザーとして、初回起動時に世界観を理解し、森と図鑑を確認したい」
```

**B) 中くらい（操作単位）** — 10〜15ストーリー：
```
「ユーザーとして、ダメカードをスワイプで仕分けたい」
「ユーザーとして、スワイプを途中でやめてもコインがもらえてほしい」
```

**C) 細かめ（アクション単位）** — 20+ストーリー：
```
「ユーザーとして、カードを右にスワイプして『やった』と記録したい」
「ユーザーとして、スワイプ後にコイン獲得アニメーションを見たい」
```

- A) 大きめ（シーン単位）
- B) 中くらい（操作単位）
- C) 細かめ（アクション単位）
- X) Other (please describe after [Answer]: tag below)

[Answer]:B

### Q4. 受け入れ基準の形式

各ストーリーの「完成の定義」をどの形式で書きますか？

**A) Given-When-Then形式**:
```
Given: ユーザーがスワイプ報告画面にいる
When: 10枚中5枚目で画面を閉じる
Then: 残り5枚分のナマケコイン（15コイン）が付与される
```

**B) チェックリスト形式**:
```
□ カードが1枚ずつ表示される
□ 右スワイプで「やった」と記録される
□ スワイプごとにコインが加算される
□ 途中で閉じても残り枚数分のコインが付与される
```

**C) 動画撮影チェック形式**（プレゼン動画に直結）:
```
撮影ポイント: スワイプ操作 → コイン獲得表示 → 途中離脱 → ボーナス表示
この一連の操作が動画で撮れればOK
```

- A) Given-When-Then形式
- B) チェックリスト形式
- C) 動画撮影チェック形式
- X) Other (please describe after [Answer]: tag below)

[Answer]:A
