# 通貨システム設計 — 質問

「怠惰を通貨にする」のロジックを完成させるための質問です。

現状の問題：
- コアメッセージ：「怠惰を通貨にする」
- 実際の設計：通貨が存在しない。ナマケモノが条件で勝手に来るだけ

修正方針：
- 通貨（仮称：ナマケコイン）を導入
- 獲得源：スワイプ報告、途中離脱、放置ボーナス
- 使い道：ここを設計する

## Question 3
通貨とナマケモノ出現の関係はどうしますか？

A) 通貨は牧場・アイテム用。ナマケモノの出現条件は今まで通り（ダメの種類・回数）で別系統
B) 通貨を使って「おもてなしアイテム」を設置 → それが出現条件の一部になる（ねこあつめのグッズに近い）
C) 一部のナマケモノは通貨で直接「招待」できる
