## Q1. 書類審査の評価基準

書類審査では何が評価されますか？（わかる範囲で）

- A) アイデアの独自性・面白さが中心
- B) 技術的な実現可能性も重視される
- C) ビジネス性・社会的インパクトも見られる
- D) 具体的な評価基準は公開されていない
- X) 自由記述

[Answer]:X ビジネス意図（Intent）の明確さ、Unit 分解の適切さ、創造性とテーマ適合性、ドキュメントの品質

第一次審査（MVP + 15分プレゼン）では何が評価されますか？（わかる範囲で）

- A) 動くプロダクトの完成度が最重要

- C) プレゼンの説得力が大きい

## Q5. MVP v1（第一次審査）のスコープ感覚

2週間で作る範囲として、あなたの感覚に近いのは？

- A) コアループが1周回る最小限（スワイプ報告 → ナマケモノ出現 → 図鑑登録）だけでいい
- B) コアループ + 通貨 + ショップまで。経済圏を体験させたい
- C) コアループ + 通貨 + ショップ + 親密度 + 手紙まで。感情的な体験まで見せたい

## Q6. デモシナリオの長さ

15分プレゼンのうち、ライブデモに使う時間の想定は？

- A) 3〜5分（残りは設計思想・ビジョン説明）
- B) 5〜7分（デモとストーリーを半々）
- C) 7〜10分（デモ中心で見せる）
- D) まだ決めていない
- X) 自由記述

[Answer]:X 動画だけを5分間流し続けるのではなく、数秒から十数秒程度の短い動画を各スライドの中に埋め込むイメージでいます

## Q7. 決勝戦で追加したい要素

決勝に進んだ場合、+3週間で最も追加したいのは？（複数選択可）

- A) ナマケモノの数を増やす（8体 → 20体以上）
- B) 演出・アニメーションの強化
- C) 親密度・手紙システムの深化

- E) 通知システムの実装
