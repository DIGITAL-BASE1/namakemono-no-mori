# コアメッセージ言語化 — 質問

あなたのロジックを整理します：

```
社会の問題：
  「生産性」「成長」「自己投資」が美徳とされている
  → 休むことに罪悪感がある
  → 普通に生きているだけで「ダメだ」と感じる人が増加

このサービスの提案：
  ダメな自分を受け入れよう
  サボることに価値を与えよう
  怠惰を通貨にしよう
```

つまり、核心は **「回復」ではなく「価値の逆転」**。
ダメを治すのではない。ダメに価値を与える。怠惰を通貨にする。

これを踏まえて、コアメッセージの言語化を一緒にやりましょう。

X) Other (please describe after [Answer]: tag below)
