# 初回〜3日目の体験設計

---

## 設計方針

> **初回オンボーディングでコアループを1周体験させる。**
> 3日目までにすべての主要機能に触れさせ、「明日も開きたい」を毎日作る。

---

## Day 1：初回起動（オンボーディング）

### フェーズ1：世界観の導入（絵本形式）

3ページの短い絵本風スライド。タップで進む。

```
【ページ1】
  "ここは「ナマケモノの森」。
   サボるほど豊かになる、ふしぎな場所。"
  （イラスト：やわらかい光が差す森。木々の間にシルエットがちらほら）

【ページ2】
  "あなたの「ダメ」に共感したナマケモノたちが、
   いつの間にか住み着くよ。"
  （イラスト：木の上からこちらを覗くナマケモノのシルエット）

【ページ3】
  "さっそく、今日のダメを教えて？"
  （→ スワイプ体験へ遷移）
```

**設計意図**:
- 3ページ以内。読むのに20秒以内
- 「ここは違う場所だよ」で安心 → 「ナマケモノが来るよ」で期待 → 「やってみよう」で行動

---

### フェーズ2：初回スワイプ体験（チュートリアル）

**枚数**: 5枚（通常の10枚より少ない。初回は軽く）

```
【カード1（チュートリアル強制）】
  🛏️ ねむねむ「二度寝した」

  操作説明として表示：
  「右にスワイプで『やった！』」
  → 右スワイプのみ受け付ける（操作説明の体で強制）
  → スワイプ成功 → 「ナマケコイン +5」の演出
  → これでネムリンの出現条件を達成（初回は条件1回で確定）

【カード2〜3】
  通常のスワイプ体験（左右どちらも可能）。
  スワイプするたびに「ナマケコイン +5」の演出。

【カード3完了時】
  画面の端に小さく足跡が出現。
  「…？ 誰かが見ているような…」

【カード4〜5】
  ここで途中離脱の誘導：
  カード5の表示時に、画面下部にゆるいテキスト表示：
  「もう疲れた？ やめてもいいよ。」
  「×」ボタンが少し目立つようになる。

  → 離脱した場合：「おつかれさま。残り○枚分のナマケコインをあげるね。」
  → 全部やった場合：「えらい！…いや、えらくなくてもいいんだけどね。」
```

**どちらのルートでも次のフェーズに進む。**

---

### フェーズ3：初めてのナマケモノ召喚

スワイプ完了（or 途中離脱）直後：

```
【演出】
  森の画面に切り替わる。
  木の上から何かが降りてくる…

  「ネムリンが あなたの森にやってきた！」

  → 図鑑登録演出（シルエット → 実体に変わるアニメーション）
  → ネムリンの一言：「zzz…あ、来ちゃった。よろしくね。」
```

**なぜネムリンか**: 睡眠系は最も普遍的なダメ。誰でも共感できる。初回確定枠として最適。

**初回確定の仕組み**: カード1が必ず睡眠系で、チュートリアルとして「やった」方向にしかスワイプできない。これにより確実にネムリンの条件を達成する。

---

### フェーズ4：ショップ体験

ネムリン召喚直後：

```
【ガイドテキスト】
  「ナマケコインが貯まったね。
   ショップで何か買ってみる？」

  → ショップ画面へ誘導

【初回限定】
  ショップに「はじめてのおやつセット」（0コイン or 極少コイン）を配置。
  → 購入のチュートリアル体験を確実にさせる

  購入後：
  「ネムリンにあげてみよう？」
  → 贈り物画面へ誘導
```

---

### フェーズ5：贈り物 → 親密度体験

```
【贈り物】
  ネムリンにおやつを贈る → 親密度ゲージが上がる演出
  ネムリンの反応：「もぐもぐ…うれしい…zzz」

【示唆】
  親密度ゲージの上に薄く表示：
  「★★★になると、ネムリンからお手紙が届くかも…」

  → 手紙コレクション図鑑のアイコンがチラッと光る（タップ不可。存在だけ示す）
```

---

### フェーズ6：森の確認 + 気配の演出

```
【森画面】
  ネムリンが木の上で寝ている。
  森の別の場所に「足跡」が表示されている。

  タップすると：
  「誰かの足跡…明日の朝、来てくれるかも？」

【オンボーディング完了メッセージ】
  「これで準備OK。
   明日の朝また来てね。ナマケモノたちが待ってるよ。」
```

---

### Day 1 夜の通知

> **初日は夜の通知を送らない。**

理由：オンボーディング内でスワイプ体験済み。同日に再度スワイプを求めるのは不自然。
翌朝のモグモグ到着通知で再エンゲージする。

## Day 2：朝の喜び + ループの実感

### 朝：通知 + 2匹目の到着

```
【朝の通知】（LINE Push通知）
  「森に新しい仲間が来ているよ！見に行こう🦥」

【ログイン時】
  森を開くと…新しいナマケモノが住み着いている！

  「モグモグが あなたの森にやってきた！」
  → 図鑑登録演出
  → モグモグの一言：「おなかすいた〜。なんか食べよ？」

【日次収穫】
  「ネムリンとモグモグが一緒に怠けて、ナマケコインを生みました！」
  → ネムリン: 8コイン + モグモグ: 6コイン = 14コイン収穫
```

**モグモグ確定の仕組み**: 図鑑#1〜5は出現条件が超緩い設計。モグモグ（#2）はオンボーディングのスワイプ内容に関わらず、翌朝に確定で到着する。

### 昼〜夕方：ショップ＆贈り物の自由体験

```
  収穫したコイン + 前日のスワイプコインで、
  ショップでアイテムを自由に購入。
  モグモグにも贈り物ができる。

  → 2匹の親密度を並行して上げる楽しさ
```

### 夜：通知 + 2回目のスワイプ

```
【夜の通知】（LINE Push通知）
  モグモグ「夜食、一緒に食べよ？」
  （ナマケモノが2匹になったので、通知のバリエーションが増えた実感）

  → タップでスワイプ報告画面へ
  → 通常の10枚スワイプ（途中離脱OK）
  → 完了後：「森に新しい足跡が…明日の朝が楽しみ。」
```

## Day 3：3匹目 + ループの定着

### 朝：通知 + 3匹目の到着

```
【朝の通知】（LINE Push通知）
  「森に新しい仲間が来ているよ！見に行こう🦥」

【ログイン時】
  Day 1〜2のスワイプ内容に応じて、条件を満たしたナマケモノが到着。
  （例：先延ばし系を複数回スワイプしていたら → アトデーノ）

  → 図鑑登録
  → 日次収穫（3匹分。コインが増えた実感）
  → ショップの品揃えが少し増える（コイン量に応じて解放）
```

### Day 3の体験ポイント

| 要素 | 体験 |
|---|---|
| 3匹目の到着 | 「自分のダメの傾向でキャラが変わる」の実感 |
| 収穫量の増加 | 3匹 → コインが増える → 経済圏の実感 |
| ショップの拡充 | 新しいアイテムが買える → 選択の楽しさ |
| 図鑑の進捗 | 3/16が埋まった。残り13のシルエットが気になる |
| ヒント2の解放 | 条件50%達成のナマケモノがいれば、ヒント2が表示される |

## 通知システム設計

### 通知の種類

| 種類 | タイミング | 内容 | 遷移先 |
|---|---|---|---|
| **朝の到着通知** | 新しいナマケモノが来た朝 | 「森に新しい仲間が来ているよ！見に行こう🦥」 | 森画面 |
| **夜のスワイプ通知** | 毎晩（ユーザー設定時刻） | ナマケモノ固有のセリフ | スワイプ報告画面 |

### 通知ルール

- **初日**: 夜の通知を送らない（オンボーディングでスワイプ済み）
- **2日目以降**: 夜の通知を毎晩送る
- **朝の到着通知**: 新しいナマケモノが来た朝のみ送る（毎朝ではない）
- **夜の通知セリフ**: 森にいるナマケモノからランダムで1匹選出。そのキャラ固有のセリフ
- **通知を無視**: ペナルティなし。放置ボーナスが溜まる

## 離脱防止メカニズム

### 1. 「明日の約束」システム
> **毎晩のスワイプ完了後に、翌朝の期待を必ず作る。**

| 状態 | 表示メッセージ |
|---|---|
| 条件達成済み（翌朝来る） | 「森に新しい足跡が…明日の朝が楽しみ。」 |
| 条件未達だが進行中 | 「足跡はまだ増えていないけど、誰かがこっちを見ている気がする…」 |
| 条件進行なし | 「ナマケモノたちが気持ちよさそうに寝ている…」 |

→ どの状態でも「明日開く理由」がある。

### 2. 日次収穫の「溜まる」演出
> **ログインしなかった日のコインも溜まる。次回ログイン時にまとめて受け取れる。**

- 「○日分のナマケコインが溜まっています！」
- 溜まったコインが一気に降ってくる演出（気持ちいい）
- 離脱してもペナルティなし。戻ってきたら「おかえり」+ ご褒美

### 3. ナマケモノの「つぶやき」更新
> **森のナマケモノをタップするたびに新しい一言をつぶやく。**

- 各ナマケモノは5種類のセリフを持つ
- タップするたびにランダムで1つ表示（前回と重複しない）
- 5種類すべて聞くと「全セリフコンプリート」の小さな達成感

### 4. 図鑑の「あと少し」表示
> **条件達成が近いナマケモノがいる場合、図鑑で視覚的に示す。**

- シルエットの周りが少し光る
- ヒント2が解放される（50%達成時）
- 「あと少しで会えそう…」の焦らし

## ダメカテゴリ再設計

### 設計原則
1. **他人に迷惑をかける行動は含めない** — キャラ開放動機で助長する恐れがある
2. **カテゴリ名はゆるく** — ゲームの世界観に合わせる
3. **カード文面は明確に** — スワイプ判断に迷わない表現

### 削除するカード

| 削除対象 | 理由 |
|---|---|
| 「仕事サボった」 | 他人に迷惑をかける |
| 「予定キャンセルした」 | 相手がいる約束を破る行為 |
| 「会議中ぼーっと」 | 仕事上の責任放棄 |
| 「返信を後回し」 | 相手を待たせる行為 |

### 新カテゴリ体系

| アイコン | カテゴリ名 | カード内容 |
|---|---|---|
| 🛏️ | **ねむねむ** | 二度寝した / 夜更かしした / 昼寝しすぎた / 布団から出られなかった |
| 🍔 | **もぐもぐ** | 深夜に食べた / ジャンクフード食べた / 食べすぎた / おやつ3回目 |
| ⏰ | **あとで〜** | 締め切りギリギリになった / やるべきことを後回しにした / 「明日やろう」と思った |
| 📱 | **ぽちぽち** | SNS見すぎた / 動画見すぎた / 無限スクロールした / 気づいたら1時間経ってた |
| 🏠 | **おうちだいすき** | 一日中ゴロゴロした / 家から出なかった / 何もしなかった / 着替えなかった |
