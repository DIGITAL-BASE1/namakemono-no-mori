# ループの深さ設計 — 質問

あなたの問いを整理します：

```
現状のループ（浅い）：
  ダメ報告 → ナマケモノが来る → 図鑑登録 → …次は？（止まる）

必要なのは：
  ダメ報告 → ナマケモノが来る → 図鑑登録 → 【ナマケモノと何かする】→ …
  この【何かする】がループを回し続ける理由になる
```

ねこあつめとの差別化の鍵もここにあります。
ねこあつめの猫は「来て、写真を撮って、去る」。関係性が育たない。
このサービスのナマケモノは「来て、住み着いて、その後がある」。

以下、「ナマケモノが来た後の体験」の方向性を提示します。

---

## Question 1
ナマケモノが来た後、ユーザーとナマケモノの間にどんな関係性を作りたいですか？

A) 育成関係 — ナマケモノを育てる（レベルアップ、進化、スキル習得）
B) 対話関係 — ナマケモノと会話する（AIで生成された会話、相談、雑談）
C) 協力関係 — ナマケモノと一緒に何かを達成する（ミッション、クエスト）
D) 観察関係 — ナマケモノの生態を観察する（日記、行動パターン、ナマケモノ同士の関係）
E) 共生関係 — ナマケモノがユーザーの生活に影響を与える（通知、アドバイス、日常のお供）
X) Other (please describe after [Answer]: tag below)

[Answer]: X この質問はもっと深堀が必要だ。Aはあり。でも進化させた後にどうなる？ただ、図鑑埋まるだけなら弱い。Bはサブ機能程度であってもいい、楽しそう。Cはちょっとピンとこないな。ミニゲームを追加するとか？Dはナマケモノ同士を組み合わせたときの化学反応って視点ではとても良い

## Question 2
「ナマケモノが来た後の体験」として、以下のアイデアの中で心が動くものはどれですか？（複数選択可）

**A) ナマケモノの物語が解放される**
- 各ナマケモノには固有のバックストーリーがある
- 親密度が上がるとストーリーが少しずつ解放される
- 「なぜこのナマケモノはここに来たのか」が語られる
- ストーリーの中にさりげない前向きメッセージが含まれる

**B) ナマケモノ同士の関係性が生まれる**
- 特定の組み合わせのナマケモノが牧場にいると、関係性イベントが発生
- 友情、ライバル、師弟関係など
- 関係性が育つと特別なシーンが見られる
- 「この2匹を一緒にするとどうなるんだろう？」という探索欲

**C) ナマケモノが牧場を変えていく**
- ナマケモノがそれぞれ牧場に影響を与える（ネムリンがいるとハンモックが生える、モグモグがいると果樹園ができる）
- どのナマケモノを集めるかで牧場の姿が変わる
- 「自分だけの牧場」が自然にできあがる

**D) ナマケモノからの贈り物**
- 親密度が上がったナマケモノが、時々アイテムや手紙をくれる
- レアなナマケモノほどレアな贈り物
- 贈り物を集めるという第2の収集要素

**E) ナマケモノに「ダメ」を教えてもらう**
- ナマケモノはそれぞれ「ダメの達人」
- ネムリンは「最高の二度寝の仕方」を教えてくれる
- モグモグは「罪悪感なく深夜に食べるコツ」を教えてくれる
- ダメを肯定するコンテンツがナマケモノから提供される

X) Other (please describe after [Answer]: tag below)

[Answer]: D 親密度ってパラメータを追加するのはいいね。各キャラには好きなもの、きらいなもの個性が設定されていて、それにあったアイテムを上げると親密度が上がるようにしたい。アイテムは集めたナマケコインでショップで買う。

E) コインの使い道はデコレーションに限定し、ナマケモノとの体験は時間経過で自然に進む
